更新文件时需要的<b>git</b> 命令 

		git add .
		git commit -m "message"  注：message 写提交的提示信息
		git pull origin gh-pages //更新版本
		git push origin gh-pages //提交版本
		输入账号
		输入密码
		即可实现文件更新上传
